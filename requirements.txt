# ===================================
# Installation Instructions
# ===================================
# 1. Install Python dependencies:
#    pip install -r requirements.txt
#
# 2. Install Ollama (separate installation):
#    Download from: https://ollama.com/download
#
# 3. Pull required AI model:
#    ollama pull gemma3:4b
#
# 4. Set up MySQL database:
#    - Install MySQL Server
#    - Create database: CREATE DATABASE Collab_DB;
#    - Run schema setup (see database/schema.sql if available)
#
# 5. Configure environment variables:
#    Create a .env file with:
#    SECRET_KEY=your-secret-key-here
#    PASSWORD=your-mysql-password

# - MySQL Server 8.0+

# ===================================
# - GloVe model auto-downloads on first use (~128MB)
# - Gemma3:4b model must be pulled manually (~2.5GB)
